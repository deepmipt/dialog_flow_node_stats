# Getting started

Installation:
```bash
# basics
pip install -e .
# run for dashboard mode:
pip install -r requirements_dashboard.txt
# run for api mode:
pip install -r requirements_api.txt
```

Examples:
```bash
# collect stats
python examples/collect_stats.py
# run dashboard:
python examples/dashboard.py
# run api:
python examples/api.py
```
